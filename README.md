# Premium Invoice Generator

A modern, responsive React application for creating professional invoices. Built with Vite, featuring a clean design that works perfectly on all devices.

## Features

- Fully responsive design for mobile, tablet, and desktop
- PWA support for mobile installation
- Professional invoice templates
- Business intelligence reports
- Receipt history management
- Print-ready invoices
- Modern UI with smooth animations

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Responsive Design

This app is optimized for all screen sizes:
- **Mobile**: Touch-friendly interface with stacked layouts
- **Tablet**: Adaptive grid systems and optimized spacing
- **Desktop**: Full feature set with expanded layouts
- **Print**: A4-sized print styles for professional output

## PWA Features

- Installable on mobile devices
- Offline-capable interface
- Native app-like experience

## Deployment

This project is ready for static deployment on platforms like Vercel, Netlify, or any web server.

- Build command: `npm run build`
- Output directory: `dist`
- Vercel config: `vercel.json`
- Netlify redirects: `public/_redirects`

For Vercel, simply import the repo and deploy. For Netlify, deploy the `dist` folder or connect the repository and use the same build command.
